Description:
	Tell the program how many exercises, from each of the following categories, 
	that you want to perform as part of a training day, and it will tell you the number of 
	distinct workouts that you could perform. 

Exercise Categories:
	Compounds - Heavy barbell lifts BESIDES Squat and Deadlift
	Assistance - Moderate bodyweight and barbell lifts to assist with Compound progress
	AccessoriesCalves - Accessory lifts that target the calves
	AccessoriesAbs - Accessory lifts that target the abdominals, particularly RA and obliques. 

Usage: (Command-Line)
	python main.py numComponds numAssistance numAccessoriesCalves numAccessoriesAbs
